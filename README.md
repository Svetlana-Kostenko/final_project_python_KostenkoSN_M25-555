# Платформа для отслеживания и симуляции торговли валютами
Платформа для отслеживания и симуляции торговли валютами позволяет в реальном времени мониторить курсы фиатных (USD, EUR, GBP и др.) и криптовалют (BTC, ETH и т. д.), используя данные из API CoinGecko (для крипты) и ExchangeRate‑API (для фиата), а также симулировать торговые операции — создавать виртуальный портфель, совершать покупки/продажи по актуальным курсам, отслеживать прибыль/убыток и анализировать историю сделок, — всё без риска реальных финансовых потерь.


Ниже представлены команды для ОС Linux. 

Необходимо настроить виртуальное окружение:  
python3 -m venv .venv  
source .venv/bin/activate  

Далее нужно выполнить 3 команды:  
make package-install,  
make install,  
make project.  

Требования: poetry, make.

# Ограничение: 18 +
